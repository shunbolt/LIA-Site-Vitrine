Le projet

LIA c’est quoi ?
LIA c'est avant tout un projet étudiant, un chatbot conçu pour les élèves du secondaire, victimes ou témoins de harcèlement scolaire. LIA se définit comme un outil supplémentaire de communication, d'échange, de transmission et signalement de problèmes entre les élèves et les autorités compétentes.

Pourquoi avoir fait ce projet ?
La grande majorité des adolescents se servent et utilisent de plus en plus d’applications internet ou mobiles pour communiquer (Réseaux sociaux – SMS – Chat). De plus, un adolescent n’a pas toujours l’aisance pour parler de ses problèmes face à un adulte souvent par peur de représailles ou de jugement de leur part.
De ces problématiques, est nés LIA, une réponse combinant ces deux aspects : à la fois une méthode de chat similaire à ce qu’utilise déjà l'adolescent et un engagement auprès de lui sans soupçon ou préjugé humain qui pourrait le troubler.

Comment cela fonctionne ?
 1. Le chatbot échange avec l’adolescent autour des problèmes qu’il rencontre ou observe à l’école ;
 2. Le bot génère une fiche et une alerte à travers les informations collectées durant la discussion ;
 3. La fiche est transmise aux autorités compétentes pour qu’ils puissent suivre le problème.

L'équipe

LIA est une équipe de huit étudiants ingénieurs au sein d’EFREI Paris. Notre équipe est composée d’étudiants en data science, en software engineering et en cloud engineering qui se sont réunis dans le cadre de nos études pour créer une solution nouvelle et durable.

Intégration

Téléchargez le fichier CSS et ajoutez ce lien dans le header. Attention vous devrez également inclure bootstrap

<link rel="stylesheet" href="chat_LIA.css">

Ensuite pour intégrer notre chatbot à votre site ajoutez le code suivant avant la fermeture de votre </body> et si vous en avez avant les <script></script>.

<div id="window">
  <div id="title_bar" class="d-flex justify-content-between">
    <div class="img_cont_banner">
      <img src="images/logoLIA_vg_C.png" class="user_img_banner">
    </div>
    <div class="flex-fill text_banner">
      <span>Parle avec LIA</span>
    </div>
    <div id="bt_top" class="btn-group" role="group" aria-label="Basic example">
      <button id="min_max_button" type="button" class="btn btn-toggle">
        <i id="icon-toggle" class="fa fa-window-minimize"></i>
      </button>
    </div>
  </div>
  <div id="box">
    <iframe id="inline_frame" src="www.lia-stop-harcelement.fr/chatbot" frameBorder="0"></iframe>
  </div>
</div>

Et enfin ajoutez cette fonction jQuery avant la fermeture de votre </body>

$(document).ready(function() {
            $("#min_max_button").click(function() {
              if ($('#icon-toggle').hasClass('fa-window-minimize')) {
                $('#icon-toggle').toggleClass('fa-window-minimize', false).toggleClass('fa-window-maximize');
              } else {
                $('#icon-toggle').toggleClass('fa-window-maximize', false).toggleClass('fa-window-minimize');
              }
              $('#box').slideToggle();
            });
          });
